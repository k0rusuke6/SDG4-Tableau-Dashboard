📌 Project Overview

This project presents an interactive Education Program & Student Distribution Analysis Dashboard designed to support SDG 4 – Quality Education. The dashboard provides insights into student participation, program performance, financial metrics, and educational accessibility across different regions and learning categories.

The analysis helps identify trends in enrollment, delivery methods, and course engagement to support data-driven educational planning and decision-making.

📂 Dataset Information

Dataset Name: Education

Domain: Education Analytics

Purpose: To analyze student distribution, education program performance, and financial outcomes across multiple dimensions.

🎯 Key Insights
Student participation varies across regions.
Online and hybrid learning modes show strong adoption.
Business and Arts courses have high enrollment.
Undergraduate programs generate the highest revenue and profit.
💡 Recommendations
Expand educational programs in low-participation regions.
Promote online learning accessibility.
Balance student enrollment across course categories and education levels.
